This game was developed from 18/02/2017 to 19/02/2017 by team Sparkings on the 2nd 99Jam at Eurekamob using Game Maker engine.
99Jam is an event promoted by 99lab, a game developers comunity, for people to make teams and create games in short time. 
On this 99Jam team Sparkings (i.e. Sparta + Vikings) was composed by:
	Carlos Eduardo da Silva - Programer
	Claudio Mota Oliveira - Programer
	Iuri Ravel Araújo - Sonoplast
	Murilo Souza - Writer/Artist
